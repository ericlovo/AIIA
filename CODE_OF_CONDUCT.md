# Code of Conduct

AIIA adopts the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)
as its code of conduct. The full text is linked above and takes precedence
over anything summarized here.

## Our pledge in short

We want AIIA to be a project where anyone can contribute — regardless of
experience, background, or identity — without being harassed, insulted, or
belittled for who they are or what they know. We expect participants to
be welcoming, patient, and respectful, and to assume good intent when
reading each other's words.

## Unacceptable behavior

- Personal attacks, trolling, insulting or derogatory comments.
- Harassment in any form, public or private.
- Posting others' private information without explicit permission.
- Sexualized language or imagery in project spaces.
- Conduct that a reasonable professional would consider inappropriate
  in a work setting.

## Scope

This code applies in all project spaces — the GitHub repository, issues,
pull requests, discussions, and any community channels — and also when
someone represents the project in public (e.g. posting as an AIIA
maintainer, speaking at an event about AIIA).

## Reporting

If you see or experience behavior that violates this code, report it
privately to **conduct@aplora.ai**. Reports will be reviewed and
investigated promptly. The maintainers will respect the privacy and
safety of the reporter.

Maintainers who do not follow or enforce this code in good faith may
face consequences determined by other project maintainers.
